# PurrCloud 退款与售后政策

## 首次订阅 7 天无理由
所有**首次订阅**的用户享有 **7 天无理由退款**：
- 起算时间：第一次包裹签收时间
- 退款金额：全额退款，PurrCloud 承担回寄运费
- 限制条件：7 天内最多开启 5 袋产品（用于试吃），其余必须保持原包装

## 冷链异常全额退
如签收时温度贴纸变色、外包装破损、食材有明显异味或变质：
- 拍照发给客服（邮箱或微信）
- **24 小时内**得到反馈
- 确认后**全额退款 + 重发一批**，不扣运费

## 猫咪不爱吃怎么办
PurrCloud 提供 **"口味不合适换配方"服务**：
- 订阅期内任意一次配送，如猫咪连续 3 顿拒食
- 可申请"换配方"，由营养师重新推荐
- 换配方 **不额外收费**，当月剩余餐食可保留或回收（回收不退款，但会返 50% 到下月账户）

## 退款时效
- 支付宝 / 微信：审核通过后 **1–3 个工作日** 到账
- 银联 / Apple Pay：**3–7 个工作日** 到账

## 以下情况不予退款
- 已签收超过 7 天的首次订阅
- 第二次及之后的常规订阅（仅支持换配方或暂停）
- 自行运输导致的损坏
- 用户填写信息错误导致的配送失败
