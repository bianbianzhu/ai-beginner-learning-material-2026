🌀 ReAct Agent Loop（完整周期）

ReAct = Reason（思考）→ Act（行动：调工具）→ Observe（观察工具结果）→ 再 Reason…
直到 AI 认为"已经可以回答用户了"，循环才退出。

1. ReasonAI 思考"我需要哪些信息？"
2. ActAI 发出 function_call
3. Observe你的代码跑函数→拿到结果→回传 function_call_output
4. Repeat / AnswerAI 看到结果，决定继续调工具 或 输出最终答案
developer (instructions)
user 提问
assistant 思考（reasoning）
function_call
function_call_output (observation)
最终 assistant 回答
📜 示例：用户问"东京和伦敦哪里更暖？"
developer · instructions
You are a weather assistant. Use the get_weather tool whenever location data is needed.
user · turn 1
Which is warmer right now, Tokyo or London?
assistant · reasoning (internal)
I need weather data for BOTH cities. I'll call get_weather twice in parallel.
assistant · function_call #1
{ name: "get_weather", arguments: { city: "Tokyo" }, call_id: "call_A" }
assistant · function_call #2
{ name: "get_weather", arguments: { city: "London" }, call_id: "call_B" }
you (the app) · function_call_output
call_id: "call_A" → { city: "Tokyo", tempC: 18, sky: "clear" }
you (the app) · function_call_output
call_id: "call_B" → { city: "London", tempC: 9, sky: "rainy" }
assistant · reasoning (internal)
Got both. 18 > 9, so Tokyo is warmer. No more tools needed — compose final answer.
assistant · final answer
Tokyo is warmer: 18°C and clear, vs London 9°C and rainy.
🔑 对应 Lesson 12 的代码循环
while (true) {
  const resp = await client.responses.create({ input: history, tools });

  const toolCalls = resp.output.filter(i => i.type === "function_call");
  if (toolCalls.length === 0) {
    // 没有更多工具调用 → 退出循环，返回最终文本
    console.log(resp.output_text);
    break;
  }

  // 有工具调用 → 本地执行 → 把结果塞回 history
  history.push(...resp.output);
  for (const call of toolCalls) {
    const result = runLocally(call.name, JSON.parse(call.arguments));
    history.push({
      type: "function_call_output",
      call_id: call.call_id,
      output: JSON.stringify(result),
    });
  }
  // 继续下一轮循环
}
